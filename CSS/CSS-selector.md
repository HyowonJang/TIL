## CSS Selector

- 태그 선택자
```
h1 {
    color : rgb(0,0,0);
}
```
- 하위 항목 선택자
```
ul em {
    color : rgb(0,0,0);
}
```
- 클래스 선택자
```
.stripe {
    color : rgb(0,0,0);
}
```
-
- ID 선택자
```
#star {
    color : rgb(0,0,0);
}
```
